# GitHub Pages Setup Instructions

## Enable GitHub Pages for QuoteVerse

Follow these steps to deploy the documentation website on GitHub Pages:

### Step 1: Go to Repository Settings

1. Visit: https://github.com/4shil/quotes-app
2. Click on **Settings** tab (top right)

### Step 2: Navigate to Pages Settings

1. In the left sidebar, scroll down to **Code and automation** section
2. Click on **Pages**

### Step 3: Configure Source

1. Under **Source**, select **Deploy from a branch**
2. Under **Branch**:
   - Select `main` from the dropdown
   - Select `/docs` from the folder dropdown
   - Click **Save**

### Step 4: Wait for Deployment

- GitHub will automatically build and deploy your site
- This takes about 1-2 minutes
- You'll see a message: "Your site is ready to be published at..."

### Step 5: Access Your Site

Your GitHub Pages site will be available at:

**https://4shil.github.io/quotes-app/**
